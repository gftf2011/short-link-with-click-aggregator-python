# short-link-python

URL shortener API (FastAPI) backed by Postgres + Redis, with a batch worker for short-code generation and a Celery-based click aggregation pipeline.

## Requirements

- Python **3.14+**
- [Poetry](https://python-poetry.org/) (the repo uses `pyproject.toml`)
- Docker + Docker Compose (for local dev stack)

## Quick start (recommended)

Bring up the full local stack via the Makefile:

```bash
make startlocal
```

This will:

- Create a local `.env` from `docker/dev/env.example`
- Start all containers from `docker/dev/docker-compose.yml`

To stop and clean up:

```bash
make stoplocal
```

|---|---|

| Redis clicks | `localhost:6381` |

## Architecture

```
Client
  │
  ▼
Nginx proxy (port 8080)
  │
  ▼
FastAPI API (Uvicorn, port 8000)
  │
  ├─ POST /v1/shortlink   → creates a short link (Postgres + Redis cache)
  ├─ GET  /v1/shortlink/{code} → redirects + fires click event to buffer
  └─ GET  /v1/clicks/{code}   → returns aggregated click count

Click pipeline:
  API → clicks:buffer (Redis list)
    └─ Celery beat (every 10s) → aggregate clicks:buffer → clicks:counts (Redis hash)
    └─ Celery beat (every 60s) → flush clicks:counts → click table (Postgres)
```

## API

All routes are mounted under `/v1`.

### Create a short link

`POST /v1/shortlink`

```bash
curl -sS -X POST "http://localhost:8080/v1/shortlink" \
  -H "content-type: application/json" \
  -d '{"url":"https://example.com","expires_at":"2026-12-31T00:00:00Z"}'
```

| Response | Description |
|---|---|
| `{"status":201,"data":{"code":"<code>"}}` | Success |
| `{"status":400,"data":{"error":"..."}}` | Validation failure |

`expires_at` is optional; omitting it defaults to ~2 weeks from now (UTC).  
`alias` is optional; providing it uses that value as the code instead of a generated one.

### Resolve / redirect a short link

`GET /v1/shortlink/{code}`

```bash
curl -sS -L "http://localhost:8080/v1/shortlink/ABC1234"
```

| Response | Description |
|---|---|
| `302` redirect to original URL | Found |
| `{"status":404,"data":{"error":"..."}}` | Not found |
| `{"status":400,"data":{"error":"..."}}` | Link expired |

### Get click count

`GET /v1/clicks/{code}`

```bash
curl -sS "http://localhost:8080/v1/clicks/ABC1234"
```

| Response | Description |
|---|---|
| `{"status":200,"data":{"short_code":"ABC1234","count":42}}` | Success |

Click counts are aggregated asynchronously and flushed to Postgres every 60 seconds by the Celery beat scheduler.

## Migrations (Alembic)

Apply all migrations:

```bash
make upgradelocal
```

Create a new autogenerated revision:

```bash
make revisionlocal m="describe change"
```

The API container automatically runs `alembic upgrade head` on startup.
